# 運用手順

## 日常起動

1. Pi5 側サーバー起動
2. AtomS3R 起動
3. Wi-Fi 接続確認
4. A ボタンで会話開始

## ボタン操作仕様

- A: 押下中録音 (PTT)
- B: キャンセル

キャンセルの効き方:

- 録音中: 有効
- 応答受信待ち: 有効
- 再生中: 有効
- HTTP 送信中: 即時中断不可

## 障害時の切り分け

1. AtomS3R が Wi-Fi 接続できるか
2. `VOICE_SERVER_URL` が正しいか
3. Pi5 サーバーが起動中か
4. Pi5 の OpenAI API キー設定が正しいか
5. 返却 WAV のフォーマットが AtomS3R 前提と合っているか

## ログ確認ポイント

- AtomS3R シリアル:
  - ボタンイベント
  - 送信開始/終了
  - キャンセル検知
- Pi5 サーバーログ:
  - リクエスト受信
  - OpenAI API 呼び出し
  - 音声返却サイズ
